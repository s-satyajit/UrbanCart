# Security Policy

## Supported Scope

This policy applies to the code in this repository, including the React frontend,
Express backend, MongoDB integration, AI endpoints, and payment flow wiring.

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive problems.

Instead:

1. Document the issue clearly
2. Include reproduction steps if available
3. Share impact details and affected area
4. Send the report privately to the project maintainer

## What to Include

- Affected file or endpoint
- Steps to reproduce
- Expected behavior
- Actual behavior
- Severity estimate
- Any suggested mitigation

## Response Goal

The project will aim to review reports promptly and coordinate a fix before public disclosure whenever possible.
